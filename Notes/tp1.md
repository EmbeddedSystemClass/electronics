###Contenu du TP 1
===

Le logiciel : [Altium Designer](http://www.altium.com/)

####Shortcuts:
====
* x -> flip horizontally
* y -> flip vertically
* space -> flip 90degres
* CTRL-SHIFT -> move component with wire;

####Procedure de base:
====

* New PCB project (default..)
  * add new schema;
    * drag and drop component
  * add New library;
    * creer des components
